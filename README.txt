Projeto Seletivo Labenu

Se trata do meu primeiro projeto na área de desenvolvimento Web. 
Projeto foi realizado para ingresso o curso de Programação Web FullStack da Escola Labenu.
Basta abrir o arquivo HTML em uma IDE como VS Code para ter acesso ao site desenvolvido.
O site tem conceitos básicos de HTML, CSS E JavaScript.
HTML para a estrutura da página, Css para a estilização da página e algumas automatizações do formulário em JS, como a função "alert" e validação de email via formulário.
